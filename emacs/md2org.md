| 기능                     | GFM Markdown                  | Org-mode                                       | 설명 / 키보드 팁                                  |      |   |      |      |      |                     |
| ---------------------- | ----------------------------- | ---------------------------------------------- | ------------------------------------------- | ---- | - | ---- | ---- | ---- | ------------------- |
| **헤더**                 | `# H1`, `## H2` … `###### H6` | `* H1`, `** H2` … `****** H6`                  | `*` 갯수 = 레벨                                 |      |   |      |      |      |                     |
| **볼드(Bold)**           | `**text**`                    | `*text*`                                       | 선택 후 `C-c C-s b` 가능                         |      |   |      |      |      |                     |
| **이탤릭(Italic)**        | `*text*`                      | `/text/`                                       | 선택 후 `C-c C-s i`                            |      |   |      |      |      |                     |
| **볼드+이탤릭**             | `***text***`                  | `*/text/*`                                     | 두 가지 강조 결합 가능                               |      |   |      |      |      |                     |
| **취소선(Strikethrough)** | `~~text~~`                    | `+text+`                                       | Org-mode +export에서 취소선 적용                   |      |   |      |      |      |                     |
| **인라인 코드**             | `` `code` ``                  | `~code~`                                       | 선택 후 `C-c C-s c`                            |      |   |      |      |      |                     |
| **코드블럭(Fenced)**       | ` ```lang ... ``` `           | `#+BEGIN_SRC lang ... #+END_SRC`               | 선택 영역 → `C-c C-, s`                         |      |   |      |      |      |                     |
| **인용(Blockquote)**     | `> quote`                     | `#+BEGIN_QUOTE ... #+END_QUOTE` <br>또는 `C-c '` | 영역 선택 후 블럭 처리                               |      |   |      |      |      |                     |
| **순서 없는 목록**           | `- item`, `* item`, `+ item`  | `- item` 또는 `+ item`                           | `TAB`으로 들여쓰기 가능                             |      |   |      |      |      |                     |
| **순서 있는 목록**           | `1. item`                     | `1. item`                                      | `TAB`으로 서브리스트                               |      |   |      |      |      |                     |
| **작업 목록(Task list)**   | `- [ ] item`, `- [x] item`    | `- [ ] item`, `- [X] item`                     | `C-c C-c`로 토글                               |      |   |      |      |      |                     |
| **표(Table)**           | `                             | col1                                           | col2                                        | ...` | ` | col1 | col2 | ...` | Org는 TAB/RET로 자동 정렬 |
| **링크(Inline)**         | `[text](url)`                 | `[[url][text]]`                                | 선택 후 `C-c C-l`                              |      |   |      |      |      |                     |
| **이미지(Image)**         | `![alt](url)`                 | `[[file:path]][[url]]`                         | Markdown export 가능                          |      |   |      |      |      |                     |
| **수평선(HR)**            | `---`, `***`, `___`           | `-----`                                        | export 시 자동 변환                              |      |   |      |      |      |                     |
| **각주(Footnote)**       | `[^1] … [^1]: text`           | `[fn:1]` → `[fn:1] footnote text`              | Org-mode inline 각주, Markdown export 시 자동 변환 |      |   |      |      |      |                     |
| **인라인 수식(Math)**       | `$inline$`                    | `\(inline\)`                                   | LaTeX 호환                                    |      |   |      |      |      |                     |
| **블럭 수식(Math)**        | `$$block$$`                   | `\[block\]`                                    | LaTeX 블럭                                    |      |   |      |      |      |                     |
| **HTML 직접 사용**         | `<tag>content</tag>`          | 동일                                             | Org export에서 HTML 가능                        |      |   |      |      |      |                     |
| **Escape 문자**          | `* [ ] ``                     | 동일                                             | 백슬래시 이스케이프 가능                               |      |   |      |      |      |                     |
| **Emoji(shortcode)**   | `:smile:`                     | 동일                                             | GitHub export 시 지원                          |      |   |      |      |      |                     |
| **@멘션**                | `@username`                   | 동일                                             | Org export 후 GitHub에서 작동                    |      |   |      |      |      |                     |
| **# 이슈/PR 참조**         | `#123`                        | 동일                                             | GitHub export 후 작동                          |      |   |      |      |      |                     |
| **Custom TOC**         | `<!-- toc -->`                | `#+TOC: headlines N`                           | N = 헤더 레벨                                   |      |   |      |      |      |                     |
| **Mermaid Diagram**    | <code>```mermaid</code>       | `#+BEGIN_SRC mermaid ... #+END_SRC`            | Org 코드블럭으로 감싸기 가능                           |      |   |      |      |      |                     |
| **HTML table**         | `<table>...</table>`          | Org 표 또는 그대로                                   | Org export에서 변환 가능                          |      |   |      |      |      |                     |
